# 🏏 DPL Cricket Dashboard

Simple cricket tournament dashboard using:

- HTML + JS (frontend)
- Google Sheets (database)
- Google Apps Script (backend API)

## Features
- Add teams and players
- Auto fixtures
- Enter match results
- Points table

## Setup

1. Create Google Sheet with:
   - Teams
   - Fixtures

2. Deploy Apps Script as Web App

3. Paste URL in script.js

4. Open index.html

## Deploy
You can host on GitHub Pages
